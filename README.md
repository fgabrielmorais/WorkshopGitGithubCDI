# 📚 Workshop de Git e GitHub

Este repositório foi criado para apoiar o aprendizado dos conceitos básicos de **Git** e **GitHub**, com atividades práticas e simples.

---

# 🎯 Objetivo

O objetivo deste workshop é ensinar:

* Inicialização de um repositório com Git
* Versionamento de arquivos
* Criação de commits
* Envio de código para o GitHub
* Edição e atualização de projetos
* Estruturação de arquivos README

---

# 🧩 Estrutura do Projeto

Este repositório está dividido em três partes principais:

---

## 1️⃣ Projetinho "Sobre Mim"

Um projeto simples em HTML (e possivelmente CSS) onde você irá:

* Editar informações pessoais
* Fazer pequenas alterações no código
* Criar commits com suas mudanças
* Enviar atualizações para o repositório

💡 Objetivo: praticar o fluxo básico do Git na prática.

---

## 2️⃣ README de Perfil

Nesta etapa, você irá criar ou editar um README voltado para perfil, que normalmente aparece na página inicial do seu GitHub.

Você pode incluir:

* Seu nome
* Uma breve descrição
* Tecnologias que está aprendendo
* Interesses ou objetivos

💡 Objetivo: aprender a usar o README como apresentação pessoal.

---

## 3️⃣ README de Documentação

Aqui você irá trabalhar com um README mais técnico, como este.

Você pode documentar:

* O que é o projeto
* Como executar
* Estrutura de arquivos
* Objetivo do código

💡 Objetivo: entender como documentar projetos de forma clara.

---

# 🚀 Fluxo que será praticado

Durante o workshop, você irá seguir este processo:

```
Editar arquivos → git add → git commit → git push
```

E, em alguns casos:

```
Fork → Clone → Editar → Commit → Push → Pull Request
```
